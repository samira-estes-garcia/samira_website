WORK

Journal Crypt

Journal Crypt is a React application with a Ruby on Rails JSON API backend. Users can CRUD private journal entries that only they have access to.

Languages Used: React/Redux Client, Ruby on Rails API, Bootstrap

WeCook!

WeCook is a rails application that allows users to upload their recipes to the website in order to share it with others. Users have full CRUD functionality with their accounts and their recipes.

Languages Used: Ruby on Rails, JavaScript, Bootstrap, OmniAuth Gem, Devise Gem

Wishlistar

WishListar is a CRUD application that allows users to create accounts where they can create and store items in their custom wishlists. WishListar helps users store any items they may desire from across the internet.

Languages Used: Ruby, ActiveRecord, Sinatra, SQL, Bulma

Daily Meals

A CLI Gem that will help the user choose what to cook for their meal.

Languages Used: Ruby
